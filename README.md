# MirrorPuzzle_Project

README

<Mirror Puzzle Project>

* 구현한 것
1. 집 내부의 맵 타일맵으로 제작함

2. 사용자 입력에 따른 플레이어 이동 구현

3. 플레이어가 맵 돌아다닐 수 있게 구현

4. 일부 퍼즐 구현
- 형 방: 거울문자 퍼즐
- 서재: 버튼 누르기 퍼즐

5. 미니게임 구현
- 베란다: 벌레 클릭해 죽이는 미니게임

6. 여러 상호작용 오브젝트 구현

7. 대화창 구현

- - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - -


* 구현할 것
※ 플레이어 이동 맵탈출 버그 고치기 ※

1. 맵의 가구 콜라이더 오류, 버그 수정
2. 기획한 퍼즐 모두 만들고 퀄리티 높이기
3. 그래픽 퀄리티 높이기 (쉐이더, 파티클, 포스트 프로세싱 등)
4. 스토리 애니메이션(시네마틱), 컷씬
5. 픽셀 퍼펙트 카메라 추가
